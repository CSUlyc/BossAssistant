## 0.4.0 - 2026-07-01

### Added

- 新增投递记录数据结构和本地存储模块。
- 支持按岗位详情链接或职位/公司/地点组合生成防重复 key。
- 支持成功投递记录的重复检测，自动跳过已成功投递岗位。
- 支持统计今日成功投递数量，并按配置的每日上限停止处理。
- 职位列表页支持受控处理候选岗位：打开岗位、查找“立即沟通”、发送多句自我介绍。
- 支持暂停、继续、停止正在运行的候选岗位处理流程。
- 失败岗位会记录失败原因并继续处理后续岗位。
- 页面显示“继续沟通/已沟通”时会补记本地记录并跳过，避免重复发送。
- 控制面板展示已投递、重复、跳过和失败结果。

### Verified

- `npm.cmd run build` 通过。
- content script 仍由 esbuild 打成独立 IIFE 文件，适合 Manifest V3 普通 content script 加载。
